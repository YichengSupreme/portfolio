GuideForge

A modular pipeline for CRISPR gRNA design automation

Project Overview

GuideForge is a modular pipeline for CRISPR gRNA choosing using UCSC Genome Browser and IDT (Integrated DNA Technologies) CRISPR tools. The pipeline supports group-specific gRNA design specifications through policy.yaml and produces a manifest log per run for reproducibility tracking.

I developed this tool after observing friends performing CRISPR gRNA design tasks manually and monotonously. The pipeline includes a configurable PAM scanner supporting multiple PAM patterns and comprehensive quality control filtering to streamline the entire workflow.

Next Steps

I'm planning to expand GuideForge with several key enhancements:

  • Multi-organism genome support: Extend the pipeline to work with genomes beyond human (hg38/hg19), supporting model organisms like mouse, zebrafish, and others commonly used in CRISPR research.
  • Second backend for sequence scoring: Add an alternative scoring backend to complement IDT's analysis, providing researchers with multiple scoring options and validation approaches.
