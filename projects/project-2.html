Emotion Recognition Benchmark Study

Assessing machine learning models for emotion prediction from EEG data

Project Overview

As the first step toward my long-term initiative, the Memory Recorder, I set out to assess the ability of current machine learning models to predict emotions from EEG data, using this project as a bridge from biochemistry into neuroscience.

While working with the DREAMER dataset, I found that the task is more challenging than some published studies suggest, which led me to reframe the project into a benchmark study.
